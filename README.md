# pdfManipulator
Basic pdf page manipulations with apache pdfBox library from console. 

## Operations allowed so far:
* 1- Merge Files

* 2- Split a file

* 3- Insert a file

* 4- Delete a page interval

* 5- Swap pages

* 6- Rotate pages

* 7- Reverse Ordering

* 8- Empty


### Note

* Needed libraries can be found at https://pdfbox.apache.org/
